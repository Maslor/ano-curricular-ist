## Instruções

1. Faz o download e extrai o zip deste repositório.

2. Certifica-te de que tens o python instalado.

3. Abre o terminal no diretório para onde extraíste o zip (ver passo 1)

4. Corre o seguinte comando: 
  `python ano-curricular-ist.py`
